Custom Analog Layout Course — Beginner Reference

NORMAL MODE — default, navigation
INSERT MODE — type text (press i)
VISUAL MODE — select text (press v)
COMMAND MODE — ex commands (press :)
🧭
Movement & Navigation
Basic Movement
h
Move left one character
l
Move right one character
j
Move down one line
k
Move up one line
Word Motion
w
Jump to start of next word
b
Jump back to start of word
e
Jump to end of current word
W
Next WORD (whitespace-delimited)
B
Back WORD (whitespace-delimited)
Line Motion
0
Start of line (column 0)
^
First non-blank character
$
End of line
g_
Last non-blank character
File Navigation
gg
Go to first line of file
G
Go to last line of file
42G
Jump to line 42 (any number)
Ctrl+d
Scroll half page down
Ctrl+u
Scroll half page up
Ctrl+f
Scroll full page forward
Ctrl+b
Scroll full page backward
zz
Center screen on cursor
zt
Scroll cursor to top
zb
Scroll cursor to bottom
Character Jump
f{c}
Find char forward on line
F{c}
Find char backward on line
t{c}
Jump before char (till)
;
Repeat last f/t/F/T
,
Reverse last f/t/F/T
%
Jump to matching bracket/paren
✏️
Editing & Operators
Undo / Redo
u
Undo last change
Ctrl+r
Redo undone change
U
Undo all changes on current line
Delete
x
Delete character under cursor
X
Delete character before cursor
dd
Delete entire line
5dd
Delete 5 lines (any count)
dw
Delete to end of word
d$
Delete to end of line
d0
Delete to start of line
dG
Delete to end of file
Change (delete + insert)
cc
Change entire line
cw
Change word
c$
Change to end of line (same as C)
C
Change from cursor to end of line
s
Substitute char under cursor
S
Substitute entire line
Copy (Yank) & Paste
yy
Yank (copy) line
5yy
Yank 5 lines
yw
Yank word
y$
Yank to end of line
p
Paste after cursor/line
P
Paste before cursor/line
Replace & Transform
r{c}
Replace one char (no insert mode)
R
Enter replace mode
~
Toggle case of character
gUw
Uppercase word
guw
Lowercase word
J
Join line below with current
.
Repeat last change — very useful!
Indent
>>
Indent line right
<<
Indent line left
=G
Auto-indent to end of file
⌨️
Insert Mode Entry
Entering Insert Mode
i
Insert before cursor
I
Insert at line start
a
Append after cursor
A
Append at end of line
o
Open new line below
O
Open new line above
ea
Append at end of word
Exiting Insert Mode
Esc
Return to Normal mode
Ctrl+[
Same as Esc (alternative)
Ctrl+c
Abort insert (no autocommands)
While in Insert Mode
Ctrl+h
Delete char before cursor
Ctrl+w
Delete word before cursor
Ctrl+u
Delete to line start
Ctrl+t
Indent current line
Ctrl+d
Un-indent current line
Ctrl+n
Autocomplete (next match)
Ctrl+p
Autocomplete (previous match)

Visual Mode & Selection
Enter Visual Mode
v
Character-wise visual select
V
Line-wise visual select
Ctrl+v
Block/column visual select
In Visual Mode
o
Move to other end of selection
aw
Select a word (with spaces)
iw
Select inner word
ab
Select block with ( )
ib
Select inside ( )
aB
Select block with { }
Actions on Selection
d
Delete selection
y
Yank (copy) selection
c
Change selection
>
Indent selection right
<
Indent selection left
~
Toggle case of selection
U
Uppercase selection
u
Lowercase selection
🔍
Search & Replace
Search
/{pattern}
Search forward for pattern
?{pattern}
Search backward for pattern
n
Next match (same direction)
N
Previous match (opposite direction)
*
Search word under cursor (forward)
#
Search word under cursor (backward)
gd
Go to local definition
Substitute (Replace)
:s/old/new
Replace first on current line
:s/old/new/g
Replace all on current line
:%s/old/new/g
Replace all in entire file
:%s/old/new/gc
Replace all with confirmation
:5,12s/old/new/g
Replace in line range 5–12
Search Options
:set hlsearch
Highlight all search matches
:noh
Clear search highlighting
:set ignorecase
Case-insensitive search
:set incsearch
Incremental (live) search
💡 Use \c in pattern for case-insensitive: /foo\c
💾
Files, Buffers & Tabs
Save & Quit
:w
Write (save) file
:w file
Save to new filename
:q
Quit (fails if unsaved)
:q!
Quit and discard changes
:wq
Save and quit
:x
Save (only if changed) and quit
ZZ
Save and quit (Normal mode)
ZQ
Quit without saving
Open & Reload
:e file
Open / edit a file
:e!
Reload file (discard edits)
:r file
Insert file contents below
Buffers
:ls
List open buffers
:bn
Next buffer
:bp
Previous buffer
:b3
Go to buffer number 3
:bd
Delete (close) buffer
Tabs
:tabnew
Open a new tab
gt
Next tab
gT
Previous tab
:tabclose
Close current tab
🪟
Windows & Splits
Split Windows
:sp
Split horizontal (same file)
:vsp
Split vertical (same file)
:sp file
Open file in horizontal split
:vsp file
Open file in vertical split
Navigate Splits
Ctrl+w h
Focus split to the left
Ctrl+w l
Focus split to the right
Ctrl+w j
Focus split below
Ctrl+w k
Focus split above
Ctrl+w w
Cycle through all splits
Resize Splits
Ctrl+w +
Increase height
Ctrl+w -
Decrease height
Ctrl+w =
Equalize all split sizes
:close
Close current split
:only
Close all other splits
🎙️
Macros & Registers
Recording Macros
q{a}
Start recording macro to register a–z
q
Stop recording macro
@{a}
Play macro from register
@@
Replay last macro
5@a
Run macro 5 times
Registers
"{a}y
Yank into register a
"{a}p
Paste from register a
:reg
Show all register contents
"0p
Paste from yank register
"+y
Copy to system clipboard
"+p
Paste from system clipboard
💡 In gvim, Ctrl+C / Ctrl+V work for clipboard too via menu

Useful Settings

:set relativenumber
Relative line numbers
:set syntax=on
Syntax highlighting on
:set tabstop=4
Tab width = 4 spaces
:set expandtab
Use spaces instead of tabs
:set wrap
Wrap long lines
:set nowrap
Disable line wrapping
:colorscheme desert
Change color theme
Marks
m{a}
Set mark at cursor (a–z)
`{a}
Jump to exact mark position
'{a}
Jump to mark's line start
``
Jump to previous position
:marks
List all marks
Misc Commands
:help {topic}
Open help for topic
:version
Show vim version info
:!
Run a shell command
Ctrl+g
Show filename and position
ga
Show ASCII value of char

y3j

c$

gU3w
Uppercase next 3 words
Text Objects — a = with surroundings, i = inner
di"
Delete text inside quotes "..."
ca(
Change a (including) parentheses
yi{

vit
Visually select inside HTML tag
